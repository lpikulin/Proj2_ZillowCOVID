Our Project Details and Documentation

  • We used a Python Flask-powered RESTful API.
  • We used Python Jupyter Notebook to clean and merge data.
  • We stored data collected in a SQLite database.

    • We decided to do our project examining the effects Covid19 has had on property values.


    The following data sets were used in our analysis:

    1. Zillow property values data, was used to measure the average change in property values from January to August 2020. We accessed data for Zillow here.
    2. New York Times data collected for Covid19 cases and deaths was used to evaluate the states, counties, and cities where residents were most affected by the virus. We accessed Covid19 data here.
    3. Census Bureau data was standardized per population, with Covid19 data to determine the number of cases/deaths in each stratum. We accessed Census data here.
    4. Rural-Urban Commuting Area (RUCA) codes data was used to classify measures of population density, urbanization, and daily commuting, to identify communities where property values were being assesed. We accessed data for RUCA here.

    Conclusions and Takeaways:

    • Overall, property values increased from January to August 2020 as COVID cases increased.
    • A time-series analysis may reveal more telling trends about spikes in COVID cases and property values. If we had more time, a time-series analysis as well as controlling for decreases in interest rates would be our next steps.
    • At the rate that prices have gone up, in certain states, it has created a housing shortage because there are less homes on the market vs demand.
